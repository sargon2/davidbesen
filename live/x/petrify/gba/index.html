VisualBoy Advance, a nice emulator: http://vboy.emuhq.com/
Jeff Frohwein (creator of the common crt0.S)'s site, a good resource: http://www.devrs.com/gba/ (especially the faq)
gbadev forums, amazingly flameless and informative: http://forum.gbadev.org/
The Pern Project, interesting (if poorly written) GBA tutorials: http://www.thepernproject.com/index2.htm
DevKit Advance, the only free way to compile code for the GBA afaik: http://devkitadv.sourceforge.net/
Socrates GBA Development Environment, an extremely large, bloated, and cross-dependant SDK: http://www.suddenpresence.com/sgade/ (good if you need to see an example of how to do something)
